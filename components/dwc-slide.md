# DWC Slide

A single slide, placed inside the **DWC Slider**'s `Slides` slot. It's a plain layout wrapper around whatever content you put inside it — it has **no configurable settings of its own**.

***

## Slots

| Slot      | Description                          |
| --------- | --------------------------------------- |
| `Content` (`default`) | The slide's content — image, text, anything. |
